Run the main file in kabasuji.builder to launch the builder.

Run the main file in kabasuji.player to launch the player.



Builder Notes:
To add extra hexominoes to the player's bullpen, drag pieces from the workspace on the
lower left corner of the screen to the palette in the bottom right of the screen.

To remove hexominoes from the player's bullpen, click on a hexomino in the palette
on the bottom right of the screen.


Builder saves level to disk when Save Button is pressed.
